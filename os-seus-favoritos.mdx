## O que é

Página com **todos os anúncios** que marcaste com o botão de coração no site. Funciona como uma "lista de desejos" pessoal para voltares facilmente aos espaços que te interessaram.

**URL**: `/favoritos`

**Onde aceder**: no dropdown do avatar, opção "Os seus favoritos".

## O que mostra

A lista de favoritos usa o mesmo layout visual da pesquisa, com o coração já preenchido em cada card para indicar que o anúncio está guardado.

- Lista visual igual à da pesquisa (cards com foto, título, preço, localização e rating)
- Coração preenchido em cada card - clicar remove dos favoritos

## Como funciona

### Marcar um anúncio como favorito

Em qualquer lado do site (cards na pesquisa, página do anúncio), o processo é o seguinte:

1. Click no coração - fica vermelho
2. Servidor regista via `POST /api/listing-like`
3. Throttle 24h por (fã, anúncio) - evita spam
4. Se passar o throttle, toast in-app aparece ao anfitrião
5. Acumula em `dailyFavoriteDigest` para email diário às 08:00

### Persistência

Os favoritos são guardados de forma diferente consoante o estado de autenticação do utilizador. Ao fazer login, os favoritos guest são migrados automaticamente para o servidor.

| Estado do utilizador | Storage |
|---|---|
| **Logged-in** | `currentUser.privateData.favorites` (server Sharetribe) |
| **Guest** (não logado) | `localStorage` chave `marketplace_favorites_guest` |
| **Login** | Migração automática guest para server |

Sincronização via `src/ducks/favorites.duck.js`.

## Notificações relacionadas

Quando alguém favorita um teu anúncio, o sistema gera duas notificações distintas com momentos diferentes.

- **Toast in-app** ao topo (poll 60s)
- **Email diário às 08:00** com resumo (top 10 anúncios favoritados)

Ver [Notificações](notificacoes-completas) e [Cron Jobs](cron-jobs).

## Diferença para "A seguir"

Favoritos e seguidores são dois sistemas distintos que guardam tipos diferentes de conteúdo. A tabela abaixo resume as diferenças de uso e comportamento.

| | Os seus favoritos | A seguir |
|---|---|---|
| **O que guarda** | Anúncios específicos | Utilizadores (anfitriões) |
| **Para que serve** | Lembrar de espaços que gostei | Acompanhar quem segues |
| **Notificação** | Email diário de favoritos | Toast quando publicam anúncio novo |

Componentes em `src/components/`. Cerca de 110 ficheiros agrupados por função. Cada componente é reutilizável em vários pontos da aplicação, garantindo consistência visual e de comportamento.

## Layout e Estrutura

Estes componentes definem a estrutura visual das páginas, controlam o scroll e gerem imagens responsivas.

| Componente | Função |
|---|---|
| `Page` | Wrapper de página com SEO, scroll behavior, scrollingDisabled |
| `LayoutComposer` | Composer de layouts (side nav / single column / etc.) |
| `LayoutSideNavigation` | Layout com sidebar (account pages) |
| `LayoutSingleColumn` | Layout simples de uma coluna |
| `AspectRatioWrapper` | Wrapper que mantém aspect ratio (fotos, vídeos) |
| `ResponsiveBackgroundImageContainer` | Background image responsivo com srcset |
| `ResponsiveImage` | `<img>` com srcset e variantes scaled |
| `Heading`, `H1`-`H6` | Headings semânticos com style |
| `OutsideClickHandler` | Fecha popup ao clicar fora |
| `KeyboardListener` | Captura teclado (Esc, setas) |

## Botões e Navigation

Componentes de ação e navegação reutilizáveis em toda a aplicação.

| Componente | Função |

| `FieldNumber` | Input numérico |

| `FieldCheckboxGroup` | Grupo de checkboxes |
| `FieldRadioButton` | Radio button |
| `FieldSelect` | Dropdown select |
| `FieldSelectTree` | Select hierárquico (categorias) |
| `FieldRangeSlider` | Slider de range (preço, etc.) |
| `RangeSlider` | Slider standalone |
| `FieldCurrencyInput` | Input com formatação de moeda |
| `FieldPhoneNumberInput` | Phone input com seletor de país |
| `FieldReviewRating` | Estrelas para review |
| `ExpandingTextarea` | Textarea que cresce com o texto |
| `LanguagesField` | Multi-select de línguas com flag-icons |
| `SocialLinksField` | Array de links sociais |
| `LocationAutocompleteInput` | Input com autocomplete Mapbox |
| `CustomExtendedDataField` | Campo dinâmico (enum/text/long/bool) |
| `CustomExtendedDataSection` | Secção de custom fields |
| `HelpText` | Texto de ajuda abaixo de field |
| `ValidationError` | Mensagem de erro de validação |

## Toasts e Banners

O sistema de toasts in-app da V1H tem 5 tipos de notificações que aparecem no canto inferior direito. O cap visual é de 6 banners simultâneos, com os restantes em fila.

| Componente | Função |
|---|---|
| `FavoriteAlert` | Toast "favoritaram o teu anúncio" |
| `FollowAlert` | Toast "começaram a seguir-te" |
| `SavedSearchAlert` | Toast "tua pesquisa tem novos resultados" |
| `NewListingNotification` | Toast "novo anúncio de quem segues" |

| `NotificationBadge` | Badge vermelho do topbar |
| `LimitedAccessBanner` | Banner permanente quando admin login-as |
| `MaintenanceMode` | Página de manutenção |

## Componentes V1H (nao vinham do Sharetribe)

Estes componentes foram criados especificamente para a V1H e não fazem parte do template base do Sharetribe. Representam as funcionalidades novas adicionadas pela equipa.

- `ExtraAlerts` - 4 novos tipos de toast (edits + reviews)
- `FavoriteAlert`, `FollowAlert`, `SavedSearchAlert`, `NewListingNotification`
- `FavoriteButton` - heart para favoritos
- `LanguagesField` - multi-select com bandeiras
- `SocialLinksField` - array de links sociais com normalização
- `RecentlyViewedSection` - carrossel
- `SimilarListingsSection` - secção bottom da listing
- `PwaInstallButton` - install PWA
- `CookieConsent` - banner GDPR custom
- `ReportListingModal`, `ReportUserModal` - denúncia
- `FeedbackPromptModal` - NPS pós-review
- `WelcomeModal` - boas-vindas
- `AddToCalendarButton` - Google Calendar
- `TranslateButton` - tradução MyMemory
- `CategoryIcons/` - 9 ícones V1H

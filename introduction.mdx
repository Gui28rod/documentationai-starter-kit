## O que é a V1H

A **Venue1Hub (V1H)** é um marketplace português que liga **anfitriões** (donos de espaços) a **clientes** (organizadores de eventos, profissionais, particulares) para arrendamento de espaços por hora, dia ou longo-prazo.

**Última atualização**: 2026-05-16
**Estado**: desenvolvimento avançado em ambiente de teste Sharetribe (`v1h4-test`). Ainda não foi para produção.

## Como funciona (resumo rápido)

O fluxo principal da plataforma envolve um cliente a encontrar e reservar um espaço publicado por um anfitrião. A V1H medeia a transação e retira uma comissão automaticamente.

1. **Cliente** pesquisa espaços por categoria, localização, preço, datas
2. Encontra um anúncio, escolhe datas/horas, paga via Stripe
3. **Anfitrião** recebe o pedido na sua [Caixa de Entrada](reservas), aceita ou recusa
4. Após a reserva, ambos podem trocar mensagens
5. Quando a reserva termina, ambos podem [avaliar](avaliacoes)
6. V1H retira **comissão** automaticamente; Stripe envia o resto ao anfitrião

## Stack técnica

- **Frontend**: React 18 + Redux Toolkit + CSS Modules
- **Backend**: Node.js + Express (renderização server-side + endpoints customizados + cron jobs)
- **Marketplace API**: gerida pela Sharetribe (sem BD própria)
- **Pagamentos**: Stripe Connect
- **Mapas**: Mapbox
- **Email**: Resend
- Detalhe completo em [Tecnologia](quickstart) e [Sharetribe vs Customizações V1H](sharetribe-vs-v1h)

## Categorias de espaços

A V1H suporta **9 categorias** próprias, cada uma com ícone SVG e filtros específicos.

| Categoria | Exemplos |
|---|---|
| Trabalho e Reunioes | Escritórios, salas de reunião, coworking |
| Educação e Cultura | Salas de aula, espaços para workshops |
| Gastronomia e Convívio | Cozinhas, salas para refeições |
| Eventos e Festas | Salões, terraços |
| Criatividade e Produção | Estúdios, ateliers, espaços de filmagem |
| Saude e Bem-estar | Salas para terapias, yoga, massagens |
| Desporto e Atividade Física | Ginásios, courts, espaços ao ar livre |
| Espaços ao Ar Livre | Quintas, terraços, jardins |
| Espaços Inusitados e Alternativos | O que não cabe nas outras |

Cada categoria tem ícone próprio (`src/components/CategoryIcons/`) + filtros específicos.

## Modelo de negócio

A V1H ganha uma comissão por cada reserva concluída. O Stripe Connect trata de toda a complexidade de KYC e payouts, incluindo conformidade com a PSD2 europeia.

- **Comissão por reserva** retirada automaticamente sobre cada transação
- Processada via **Stripe Connect** (V1H recebe comissão, anfitrião recebe o resto)
- Percentagem configurada no Sharetribe Console (pendente confirmação para go-live)
- Ver [Pagamentos](pagamentos) para detalhe técnico

## Detalhes especiais

### PageBuilder (sem rota própria)
O PageBuilder é o motor universal que renderiza páginas cujo conteúdo é definido no Sharetribe Console, sem necessidade de programação. As chefes da V1H podem editar texto, imagens e secções diretamente no Console.

- Lê asset (`pageAsset.{name}`)
- Mapeia secções (`SectionHero`, `SectionCarousel`, etc.) para componentes
- Suporta SEO schema + layouts customizados
- Usado por `LandingPage`, `CMSPage`, `FooterContainer`

### PreviewResolverPage
Quando um asset CMS aponta para `content/pages/about`, esta página redireciona para `/sobre-nos`. Útil para preview de mudanças antes de publicar.

### StyleguidePage
Página oculta de desenvolvedor, acessível em `localhost:3000/styleguide`. Mostra todos os componentes, ícones, cores, tipografia. Útil para designers e devs verem o sistema visual completo sem percorrer o site.

### Wrappers (sem rota)
- **TopbarContainer** - renderiza Topbar em todas as páginas
- **FooterContainer** - renderiza Footer em todas as páginas
- **PageBuilder** - usado dentro de outros containers

## Por origem

### Vindos do Sharetribe (base do template)
Estes containers são herdados do Sharetribe Web Template e foram adaptados com pequenas customizações visuais e de conteúdo.

- AuthenticationPage, CMSPage, CheckoutPage, ContactDetailsPage
- EditListingPage, EmailVerificationPage, InboxPage, LandingPage
- ListingPage, MakeOfferPage, ManageAccountPage, ManageListingsPage
- NoAccessPage, NotFoundPage, PageBuilder, PasswordChangePage
- PasswordRecoveryPage, PasswordResetPage, PaymentMethodsPage
- PreviewResolverPage, ProfilePage, ProfileSettingsPage
- RequestQuotePage, SearchPage, StripePayoutPage, StyleguidePage
- TransactionPage

### Customizados pela V1H
Estes containers foram criados de raiz ou substituídos na totalidade para servir as necessidades específicas da plataforma Venue1Hub.

- **AboutPage** - página própria sobre nós
- **ComoFuncionaPage** - como funciona com FAQ
- **ContactPage** - formulário V1H
- **CookiePolicyPage**, **PrivacyPolicyPage**, **TermsOfServicePage** - atualizados PT/EN
- **DestacaAnuncioPage** - destacar anúncio
- **DirectMessagePage** - chat direto melhorado
- **FavoritesPage**, **FollowingPage**, **SavedSearchesPage**, **RecentlyViewedPage** - novas features V1H

Total: **27 vindos do Sharetribe** + **14 customizados pela V1H** = 41 containers.

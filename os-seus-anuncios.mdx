## O que é

Página onde o anfitrião gere **todos os seus anúncios** (publicados, rascunhos, fechados, pendentes de aprovação).

**URL**: `/your-listings`
**Ficheiro**: `src/containers/ManageListingsPage/`
**Onde aceder**: dropdown do avatar → **"Os seus anúncios"** *(só aparece se tens pelo menos 1 anúncio, mesmo em rascunho)*

## O que mostra

Cada anúncio aparece como **card**:
- Foto principal
- Título
- Preço
- Localização
- Rating (com IconReviewStar)
- **Badge de estado**:
  - 🟢 **Published** — visível no marketplace
  - 🟡 **Pending Approval** — admin tem de aprovar
  - 🔘 **Draft** — em rascunho, ainda não publicado
  - ⚫ **Closed** — escondido (mas dados preservados)

## Ações por anúncio (menu dropdown ⋯)

| Ação | O que faz |
|---|---|
| **Edit** | Abre o `EditListingWizard` |
| **Close** | Esconde do marketplace (sem deleção real) |
| **Open** | Republica anúncio fechado |
| **Discard draft** | Elimina rascunho (com modal de confirmação) |
| **Delete** | Fecha + adiciona a `deletedListingIds` (localStorage + privateData server) |

## HostDashboard

Componente complementar que mostra **stats do anfitrião** na mesma página:

- Total de bookings no mês
- Revenue acumulado
- Pending payouts (à espera de transferência Stripe)
- Rating médio dos anúncios
- Reviews recentes

Dados via `GET /api/host-stats` ([Endpoints](endpoints)).

## Flow de deleção

1. Click "Delete" → `addDeletedListingId(id)` (localStorage)
2. `onCloseListing(id)` (Sharetribe SDK)
3. Sync `deletedListings` para `privateData` do user (server)
4. Filtering local remove do view **sem reload**

## Paginação

Se tiveres muitos anúncios (>20), aparece paginação no fundo (1, 2, 3, ...).

Para mais detalhe técnico (ficheiro fonte, helpers, etc.) ver [Catálogo de Containers](catalogo-containers).

## O que é

Página onde editas o **teu perfil público**, incluindo avatar, bio, línguas faladas e links para redes sociais. As informações editadas aqui ficam visíveis para qualquer pessoa que visitar o teu perfil.

**URL**: `/profile-settings`

**Onde aceder**: no dropdown do avatar, opção "Configurações do perfil".

## Campos editáveis

### Form: `ProfileSettingsForm`

O formulário de perfil permite editar todos os dados públicos da conta. Cada campo tem validação específica para garantir que a informação apresentada ao público está formatada corretamente.

| Campo | Validação |
|---|---|
| **Avatar** (foto) | Upload com transient state |
| **First name** | Letras + acentos + apóstrofos + hífens, máx 50 chars |
| **Last name** | Idem |
| **Display name** | Alfanumérico + símbolos, máx 60 chars |
| **Bio** | Textarea, máx 500 chars |
| **Localização** | Autocomplete Mapbox |
| **Línguas faladas** | Multi-select com bandeiras (`LanguagesField`) |
| **Social links** | Array de URLs com normalização (`SocialLinksField`) |

### Línguas faladas (LanguagesField)

O campo de línguas faladas usa um componente multi-select com bandeiras de cada país/língua, permitindo ao anfitrião comunicar em que idiomas consegue atender os clientes.

Lista com bandeiras (biblioteca `flag-icons`):
- Português
- English
- Español
- Français
- Italiano
- Deutsch
- E mais

### Social Links (SocialLinksField)

O campo de links sociais suporta as principais redes sociais. A URL é normalizada automaticamente, pelo que o utilizador pode introduzir apenas o nome de utilizador e o link completo é gerado.

Plataformas suportadas:
- Facebook
- Instagram
- Twitter
- LinkedIn
- TikTok
- YouTube
- Pinterest

URL normalization automática (ex: `@user` passa a `https://instagram.com/user`).

## Diferença para "Configurações da conta"

"Configurações do perfil" é a área pública do utilizador. "Configurações da conta" é a área privada com credenciais e pagamentos.

| | Configurações do perfil | Configurações da conta |
|---|---|---|
| **Edita** | Dados públicos (bio, foto, social) | Dados privados (email, password, pagamento) |
| **URL** | `/profile-settings` | `/account/*` |

## Os Seus Favoritos (FavoritesPage)

A única página listada aqui que não está aprofundada noutra doc.

**URL**: `/favoritos`
**Ficheiro**: `src/containers/FavoritesPage/FavoritesPage.js`

### Funcionamento

- Lista visual igual à da pesquisa (cards com foto + título + preço)
- **Coração já preenchido** em cada card → click remove dos favoritos
- Lista carregada de `currentUser.privateData.favorites`
- Sincronizada via [`favorites.duck.js`](catalogo-ducks)

### Persistência

| Estado do user | Storage |
|---|---|
| Logged-in | `currentUser.privateData.favorites` (server Sharetribe) |
| Guest | `localStorage` chave `marketplace_favorites_guest` |
| Login | Migração automática (guest → server) |

### Notificações relacionadas

Quando alguém favorita um anúncio:
1. `POST /api/listing-like` regista
2. Throttle 24h por (fã, anúncio)
3. Se passar throttle → toast in-app no anfitrião
4. Acumula em `dailyFavoriteDigest` para email às 08:00

Ver [Notificações](notificacoes-completas) + [Cron Jobs](cron-jobs).

## Terminar Sessão

Não é página — é **ação Redux**:

1. Click → `dispatch(logoutThunk())`
2. Limpa token Sharetribe
3. Limpa `currentUser` do Redux state
4. Limpa caches (favorites, follow, etc.) que estavam em memória
5. Redirect para homepage (`/`)

## Condicionais

Alguns itens só aparecem se condições forem cumpridas:

| Item | Condição |
|---|---|
| Destacar Anúncio | Tem pelo menos 1 listing **publicado** |
| Os seus anúncios | Tem pelo menos 1 listing (mesmo em rascunho) |
| Pesquisas guardadas | Sempre (mas vazia se ainda não guardou nenhuma) |
| A seguir | Sempre (mas vazia se ainda não segue ninguém) |

## ⚙️ Plataforma

### Como Funciona

| | |
|---|---|
| **URL** | `/como-funciona` |
| **Componente** | `src/containers/ComoFuncionaPage/ComoFuncionaPage.js` |
| **Conteúdo** | Hardcoded em React com i18n (PT/EN) |

**Estrutura**:
- 2 secções principais: "Para Organizadores" e "Para Anfitriões"
- **Steps ilustrados** — 4 passos por audiência (escolher espaço → reservar → confirmar → desfrutar)
- **Garantias de segurança** — 3 cards (pagamento seguro, verificação, suporte)
- **FAQ** integrado no fundo da página (`FAQContent.js`) com accordions expansíveis
- **Scroll automático** para FAQ via hash `#faq` (link direto navega já aberto)
- **CTAs** para "Pesquisar Anúncios" e "Publicar o Meu Espaço"

**Funcionalidades especiais**:
- Ilustrações SVG inline (vetoriais)
- I18n completo PT/EN
- Anchor links funcionais

### Definições de Cookies

**Não é página** — é o **modal CookieConsent**.

Carregar no link "Definições de cookies" do footer:
1. Dispara `window.__v1hOpenCookiePrefs()`
2. Abre o modal `CookieConsent` em modo modal
3. Utilizador pode alterar consentimento por categoria

Ver [Cookies & Privacidade](cookies-privacidade) para detalhe técnico.

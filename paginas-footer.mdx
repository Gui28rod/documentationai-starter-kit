As páginas legais e institucionais são as que estão linkadas no rodapé (footer) do site. Incluem páginas obrigatórias por lei (Termos de Serviço, Política de Privacidade, Política de Cookies) e páginas informativas (Como Funciona, Sobre Nós, Contacto, FAQ). Estão agrupadas em três colunas no footer: Plataforma, Empresa e Suporte. Esta página descreve uma a uma, indicando para que serve cada página e como o utilizador a usa.

## Plataforma

### Como Funciona

| | |
|---|---|
| **URL** | `/como-funciona` |
| **Componente** | `src/containers/ComoFuncionaPage/ComoFuncionaPage.js` |
| **Conteúdo** | Hardcoded em React com i18n (PT/EN) |

A página "Como Funciona" apresenta o marketplace para as duas audiências principais, explicando o processo de cada uma com steps ilustrados e garantias de segurança.

**Estrutura**:
- 2 secções principais: "Para Organizadores" e "Para Anfitriões"
- **Steps ilustrados** - 4 passos por audiência (escolher espaço, reservar, confirmar, desfrutar)
- **Garantias de segurança** - 3 cards (pagamento seguro, verificação, suporte)
- **FAQ** integrado no fundo da página (`FAQContent.js`) com accordions expansíveis
- **Scroll automático** para FAQ via hash `#faq` (link direto navega já aberto)
- **CTAs** para "Pesquisar Anúncios" e "Publicar o Meu Espaço"

**Funcionalidades especiais**:
- Ilustrações SVG inline (vetoriais)
- I18n completo PT/EN
- Anchor links funcionais

## Empresa

### Sobre Nós

| | |
|---|---|
| **URL** | `/sobre-nos` |
| **Componente** | `src/containers/AboutPage/AboutPage.js` |

A página "Sobre Nós" apresenta a equipa e os valores da V1H. O conteúdo é hardcoded em React com suporte PT/EN completo.

**Estrutura**:
- **Hero** com imagem de fundo e slogan

**I18n**: completo PT/EN.

### Contacto

| | |
|---|---|
| **URL** | `/contact` |
| **Componente** | `src/containers/ContactPage/ContactPage.js` |

O formulário de contacto envia os dados via `POST /api/contact` para o email de admin, com validação imediata no cliente.

**Form**:
- Nome
- Email
- Telefone (com **seletor de país** e bandeira, biblioteca `flag-icons`)
- Assunto
- Mensagem
- Submit - `POST /api/contact` - email para `admin@v1h.net`

**Países suportados** no seletor de phone prefix:
PT, ES, FR, GB, DE, IT, NL, BE, CH, US, BR, AO, MZ

**Info cards na lateral**:
- Email: `admin@v1h.net`
- Endereço: Mira (Portugal)
- Telefone: `+351 930 432 478`
- Horário: Segunda a Sexta, 9h–18h

**Validações**: regex customizada para cada campo com feedback imediato no frontend.

### Definições de Cookies

**Não é página** - é o **modal CookieConsent**.

Carregar no link "Definições de cookies" do footer:
1. Dispara `window.__v1hOpenCookiePrefs()`
2. Abre o modal `CookieConsent` em modo modal
3. Utilizador pode alterar consentimento por categoria

Ver [Cookies & Privacidade](cookies-privacidade) para detalhe técnico.

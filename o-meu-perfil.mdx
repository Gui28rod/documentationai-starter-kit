## O que é

A página pública do teu perfil, visível para qualquer pessoa (logged in ou guest) quando clica no teu nome num anúncio, numa review ou em qualquer outro sítio do site. É a tua "vitrine" perante a comunidade.

**URL**: `/u/:userId`

**Onde aceder**: dropdown do avatar (canto superior direito), na opção "O meu perfil".

## O que mostra

### Sidebar (lateral)
- **Avatar grande** (foto de perfil)
- **Botão Seguir / A seguir**, para outros poderem seguir o teu perfil
- **Rating** e número de avaliações recebidas como cliente
- **Followers count**, quantas pessoas te seguem
- **Línguas faladas** (com bandeiras)
- **Social links** (Facebook, Instagram, etc.)
- **Response time** (por exemplo, "Responde em ~2h")
- **Last online** ("Online agora", "há X min", "há X dias")
- **Localização**

### Conteúdo principal
- Os teus anúncios publicados (com `ProfileListingsSearchBar` para outros filtrarem)
- Reviews que recebeste
- A tua bio

## Diferença para "Configurações do perfil"

A página de perfil público e a página de configurações de perfil servem propósitos distintos. A tabela abaixo resume a diferença principal.

| | O meu perfil | Configurações do perfil |
|---|---|---|
| **Para que serve** | Ver o teu perfil público | Editar o teu perfil |
| **Quem vê** | Toda a gente | Só tu |
| **URL** | `/u/:userId` | `/profile-settings` |

Detalhe completo: [Perfis de Utilizador](perfis).

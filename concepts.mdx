## Navigation shape (sample)

The navigation structure is defined in `documentation.json`. Each group contains pages with a title and a path that maps to an `.mdx` file in the repository.

```json
{
  "tab": "Documentation",
  "groups": [
    {
      "group": "Getting Started",
      "pages": [
        { "title": "Introduction", "path": "introduction" }
      ]
    }
  ]
}
```

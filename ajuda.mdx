## O que é

Página com **formulário de contacto** + **perguntas frequentes** (FAQ) para utilizadores tirarem dúvidas.

**URL**: `/contact`
**Ficheiro**: `src/containers/ContactPage/`
**Onde aceder**:
- Dropdown do avatar → **"Ajuda"**
- Ícone (?) no topbar

## Form de contacto

Campos pedidos:

| Campo | Validação |
|---|---|
| **Nome** | Obrigatório |
| **Email** | Formato email válido |
| **Telefone** | Com seletor de país + flag (biblioteca `flag-icons`) |
| **Assunto** | Texto |
| **Mensagem** | Textarea |

**Submit** → `POST /api/contact` → envia email para `admin@v1h.net` (configurável via env `CONTACT_RECIPIENT`).

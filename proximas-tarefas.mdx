# Próximas Tarefas Sugeridas

Ordem prática para finalizar o projeto V1H e preparar o go-live.

## 1. Decidir e integrar chatbot

Avaliar **AnyChat** (`app.anychat.one`) ou alternativas (Crisp, Tidio). Quando as chefes decidirem, basta enviar o snippet `<script>` da plataforma escolhida para integrar no `Page.js`.

## 2. Link Substack no footer

Adicionar link para a newsletter da V1H (`venue1hub.substack.com`) na secção apropriada do footer.

## 3. Documentation.AI público

Decidir se criar `help.venue1hub.com` (site de FAQ público) com base nas dúvidas mais comuns dos utilizadores.

## 4. Limpar pendências legais

Atualizar Termos e Política de Privacidade com os dados completos da empresa (NIF, capital social, registo, DPO, comissão). **Bloqueante** para go-live.

## 5. Decisão final mobile app

Decidir entre PWA (sem custo) e Capacitor (€124 + lojas de apps + Apple Developer).

## 6. Switch para produção

Quando tudo o resto estiver pronto:

- Stripe: trocar test keys por live keys
- Integration SDK: trocar para credenciais do ambiente Live Sharetribe
- GA4: clicar "Copy changes to live" no Console Sharetribe
- DNS: apontar `venue1hub.com` para o servidor de produção
- Verificar que todos os emails e cron jobs funcionam em produção

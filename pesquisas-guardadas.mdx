## O que é

Página com **todas as pesquisas** que guardaste — útil para repetir buscas frequentes ou receber notificação quando há novos anúncios que coincidem.

**URL**: `/pesquisas-guardadas`
**Ficheiro**: `src/containers/SavedSearchesPage/`
**Onde aceder**: dropdown do avatar → **"Pesquisas guardadas"**

## Como guardar uma pesquisa

1. Vai à página de pesquisa (`/s`) e aplica filtros (categoria, localização, preço, etc.)
2. Carrega no **botão coração ❤️** ao lado da contagem de resultados
3. Pesquisa fica guardada (a vermelho = guardada / a branco = não guardada)
4. Aparece nesta página

## O que mostra

Cada pesquisa guardada aparece como **card** com:
- **Label** (descrição da pesquisa, ex: "Eventos em Lisboa < 100€")
- **Filtros aplicados** (categoria, localização, preço, etc.)
- **Botão "Repetir pesquisa"** → vai para `/s` com os filtros
- **Botão "Remover"** → elimina da lista

## Dedup canónico

A V1H **detecta duplicados** automaticamente:
- A mesma pesquisa não é guardada 2x
- Comparação por "chave canónica" (params normalizados sem `page`, `sort`, `mapSearch`, `bounds`)
- Se tentas guardar duplicado → toggle remove em vez de criar

## Notificações de novos resultados

A cada **5 minutos**, o cron `notifySavedSearchesJob` verifica se há **anúncios novos** que correspondem às tuas pesquisas:

- Se sim → **email**: "X novos resultados para a tua pesquisa Y"
- Top 6 anúncios novos + link para "Ver todos"
- Também aparece **toast in-app** ao abrir o site

## Limite

Máximo **25 pesquisas guardadas** por utilizador. Se atingires, tens de remover alguma para guardar nova.

## Persistência

- Server-side em `currentUser.privateData.savedSearches`
- Não há versão guest — só disponível para logged-in users
